# GrowStack

## Overview

GrowStack is a luxury, high-end landing page concept for a premium digital growth and conversion engineering studio. The project includes a fully styled homepage with marketing sections, a live ROI calculator, pricing toggle, contact form, and interactive animations.

## Key Features

- Hero section with bold brand messaging and visual focus
- Marquee-style trusted brand panel
- Services / ecosystems overview with premium copy and cards
- Interactive ROI calculator for monthly traffic and conversion modeling
- Pricing toggle between monthly and annual billing
- Contact inquiry form with animated success state
- Newsletter signup interaction
- Smooth animations powered by GSAP

## Included Files

- `index.html` - Primary landing page markup and inline scripts
- `style.css` - Custom styles for the landing page design
- `images/hero.png` - Hero visual asset
- `images/service.png` - Service section visual asset

## How to Use

1. Open `index.html` in your browser to view the landing page.
2. Use a local static server for faster development and asset loading if desired.

### Optional local server example

- Python 3:
  - `python3 -m http.server 8000`
- Node.js:
  - `npx http-server .`

Then visit `http://localhost:8000`.

## Notes

- The ROI calculator uses an assumed average order value of ₹2,500.
- The pricing toggle switches between `₹149,999 /month` and `₹119,999 /month, billed annually`.
- Navigation and mobile menu behavior are implemented with vanilla JavaScript.
